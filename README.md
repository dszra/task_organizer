A task management application built with Django on the server side and Fetch/AJAX on the frontend.
1. The application supports adding tasks, including directly from the homepage
2. setting the display order of tasks
3. deleting tasks
4. selecting a task to work on “now,” which automatically moves it to the top of the list
5. Adding comments to tasks
6. Editing existing tasks.

The next steps:
1. Adding subtasks with the ability to mark them as completed
2. Introducing task priorities that will determine the task’s display color
3. Allowing users to set task deadlines with visual indicators when a deadline is approaching
4. Ability to create projects and assign tasks to them
5. Filtering tasks by project and due date
6. Further UI improvements
7. The task currently being worked on displayed in a clearly visible section of the application.
